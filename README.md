# Download Sorter

A small Chrome extension that helps sort downloaded files automatically and improve file management.

## Features

- Automatically organizes downloads based on rules
- Simple extension UI
- Lightweight and easy to use

## Installation

1. Open `chrome://extensions/` in Chrome.
2. Enable **Developer mode**.
3. Click **Load unpacked**.
4. Select this extension folder.

## Usage

- Click the extension icon to open the popup.
- Configure your download sorting preferences.
- Downloads will be sorted according to the extension rules.
